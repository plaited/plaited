Function useWorker

Creates a type-safe interface for Web Worker communication with automatic cleanup. Integrates workers with Plaited's event system for seamless messaging.

Example: Basic Usage

test('userWorker|defineWorker: send and receive', async () => {
  const spy = sinon.spy()
  const send = useWorker((evt: BPEvent) => spy(evt), `${import.meta.dir}/worker.ts`)
  send({
    type: 'calculate',
    detail: { a: 9, b: 10, operation: 'multiply' },
  })
  await wait(200)
  expect(
    spy.calledWith({
      type: 'update',
      detail: 90,
    }),
  ).toBeTruthy()
})

  • Type-safe message passing
  • Automatic event handling
  • Resource cleanup
