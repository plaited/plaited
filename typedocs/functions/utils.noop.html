Function noop

No-operation (no-op) function that does nothing and returns nothing. Useful as a default callback, stub, or placeholder function.

Features:

  • Accepts any number of arguments (typed via generic)
  • Type-safe with TypeScript
  • Zero impact on memory/performance
  • Void return type

Example

// As a default callback
const callback = props.onChange ?? noop;

Prefer this over empty arrow functions or undefined checks for better performance and cleaner code
