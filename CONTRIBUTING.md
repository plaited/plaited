# Contributing to Plaited

**Plaited is open-source, not open-contribution.**

If you are reading this on GitHub or some other Git repository or service, then you are looking at a mirror. While the Plaited source code is managed using Git, it is done as a submodule of our Plaited Pro offering.

## Reporting Issues

The preferred way to report bugs is to visit [GitHub Issues](https://github.com/plaited/plaited/issues).

## Questions and Discussions

For questions, feature requests, or general discussion about Plaited, visit [Plaited Discussions](https://github.com/orgs/plaited/discussions).
