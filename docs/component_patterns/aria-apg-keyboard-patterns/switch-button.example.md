Switch Example Using HTML Button
================================

Keyboard Support
----------------

| Key | Function |
| --- | --- |
| Tab | 
*   Moves keyboard focus to the `switch`.

 |
| Space, Enter | 

*   Toggle switch between on and off.

 |
