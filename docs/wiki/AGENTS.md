# AGENTS.md

## Wiki Rules

This directory is a repo-local architecture wiki. It is lower authority than
`src/`, tests, root `AGENTS.md`, and applicable skills.

Use wiki pages for synthesis, lineage, and current conceptual doctrine. Do not
promote wiki-only event names, schemas, actor APIs, or module names into runtime
contracts unless they also exist in source or tests.

When updating this wiki:

- Keep pages short enough for agents to read in one pass.
- Prefer many focused pages over one large essay.
- Link related pages explicitly.
- Record meaningful conceptual changes in `log.md`.
- Preserve source/provenance notes when summarizing historical references.
- Call out implementation gaps when doctrine and source have not converged.
