Example Disclosure (Show/Hide) for Image Description
====================================================

Keyboard Support
----------------

| Key | Function |
| --- | --- |
| Tab | Moves keyboard focus to the disclosure button. |
| Space or  
Enter | Activates the disclosure button, which toggles the visibility of the long description. |
