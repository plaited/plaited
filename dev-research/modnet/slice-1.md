# Slice 1

## Target

Introduce Sovereign Exchange Attempt-DAG TeamHub concepts for inter-node collaboration.

## Scope

- modnet research lane only
- A2A artifact exchange concepts
- no replacement of local sovereign node design

## Required

- git bundles modeled as A2A artifacts
- import/export/apply remains PM-governed
- attempt lineage and provenance travel with exchanged artifacts
- supports enterprise/shared-service node themes

## Preserve

- sovereign node boundaries remain intact
- A2A stays the inter-node exchange boundary
- PM remains import/export/apply authority
- local node git and memory strategy are not replaced by a shared hub

## Avoid

- direct free-push hub semantics inside a node
- bypassing constitution or boundary policy
- replacing A2A with an external hub protocol

## Acceptance Criteria

- bundle artifacts are framed as A2A-governed exchange objects
- provenance requirements for exchanged artifacts are explicit
- the design preserves node sovereignty and PM authority
